Invadem: A simple shoot'em up game written in Java

21 Oct 2019

Background:

Invadem is the FIRST project I completed independently since I started studying Software Engineering in 2019. It is the second assignment of the unit INFO1113 in USYD, and I have obtained full mark for this assignment.

It is a simple shoot'em up game written in Java, where the player controls a tank that fights off a horde of invading space ships. I have used the Processing library within your project to allow you to create a window and draw graphics. It is worth to mention that the Processing Library is not compatible with Java 9+.

The project contains a number of entities:

  • Tank - Player
  • Invader - Monster
  • Barrier
  • Projectile
