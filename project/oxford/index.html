Oxford Dictionary LITE

14 May 2021

Background:

Oxford Dictionary LITE is a software I developed to experiment with integrating third-party APIs. More importantly, it is the pre-work for the final exam of SOFT3202 in the University of Sydney.

Oxford Dictionary Lite is a simple software developed based on the official thesaurus information API provided by Oxford Dictionary.

Frankly speaking, this dictionary is far less efficient and convenient than the website of the official Oxford Dictionary, or apps such as Youdao Translation.

My original intention of developing it is on the one hand, it is one of my assignment requirements from the university, and the other is to test the feasibility of requesting external APIs through HTTP.

More importantly, parsing the JSON object of thesaurus information is very challenging in my opinion, because the content of different vocabulary queries may be quite different. For example, some words have no example sentences, and some words have no synonyms, so it takes a lot of effort to process.

Key Implementations:

The software development involves the following key implementation features:

  • Test Driven Development (TDD)
  • Database - SQLite-JDBC
  • HTTP & JSON
  • Concurrency - SwingWorker
  • External APIs:
    1. Input API: Oxford Dictionary Developer API
    2. Output API: SendGrid API
